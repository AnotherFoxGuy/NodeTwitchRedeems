Help

Here is some insight of the available commands and keywords to use in the actions parameters.

Commands

  • createAudioSource

    • Creates a new audio source

  • createColorFilter

    • Creates a new color adjustment (hue shift) filter

  • createFreezeFilter

    • Creates a new freeze filter. Requires this plugin: OBS Freeze Filter

  • createScrollFilter

    • Creates a new scroll filter

  • deleteFilter

    • Delete a filter from a source

  • deleteSource

    • Delete a source from a scene

  • displayFilter

    • Displays/hide a source filter. Filter has to already exists

  • displaySource

    • Displays/hide a source

  • saveScreenshot

    • Saves a screenshot of the specified source

  • setFilterDelay

    • Sets the time warp scan filter delay. Requires this plugin: OBS Time Warp Scan

  • setFilterRotation

    • Sets the time warp scan filter rotation. Requires this plugin: OBS Time Warp Scan

  • setSourcePosition

    • Sets the position of the specified source

  • setSourceScale

    • Sets the scale of the specified source

  • wait

    • Waits for x seconds. Specified in ms. (1s = 1000ms)

Keywords

  • '$obsSceneName'

    • Fetches the name of the current OBS scene at the moment the action is triggered

  • $obsXScale

    • Fetches the X scale of the specified source. Relevant commands: 'setSourceScale', 'setSourcePosition'

  • $obsYScale

    • Fetches the Y scale of the specified source. Relevant commands: 'setSourceScale', 'setSourcePosition'

  • $obsXPosition

    • Fetches the X position of the specified source. Relevant commands: 'setSourcePosition', 'setSourceScale'

  • $obsYPosition

    • Fetches the Y position of the specified source. Relevant commands: 'setSourcePosition', 'setSourceScale'

  • $obsWidth

    • Fetches the obs source width. Relevant commands: 'setSourceScale', 'setSourcePosition',

  • $obsHeight

    • Fetches the obs source height. Relevant commands: 'setSourceScale', 'setSourcePosition'

  • $obsCanvasWidth

    • Fetches the obs canvas width. Relevant commands: 'setSourcePosition', 'setSourceScale'

  • $obsCanvasHeight

    • Fetches the obs canvas height. Relevant commands: 'setSourcePosition', 'setSourceScale'

  • $varRandom

    • Returns a random value following the Javascript Math.random() function

Tips

  • You can use and/or combine Math functions and operators

    • Examples:
    Math.floor($varRandom) * 360
    $obsXScale / 2 + 15

  • X and Y in the 'setSourceScale' and 'setSourcePosition' commands can be blank

    • It just won't modify anything

  • Scenes and sources names are case sensitive
